# <a name="example-scenes"></a>示例场景

MRTK 提供各种类型的示例场景，这些场景演示 MRTK 的功能和构建基块以实现空间用户体验。 遇到和二级示例场景有助于理解这些功能并将其应用于项目。 

## <a name="how-to-acquire-example-scenes"></a>如何获取示例场景

### <a name="using-mixed-reality-feature-tool-and-unity-package-manager"></a>使用混合现实功能工具和 Unity 包管理器

可以通过 [混合现实功能工具](/windows/mixed-reality/develop/unity/welcome-to-mr-feature-tool)来下载和导入 **混合现实 Toolkit 示例** 包

<img src="features/images/hand-interaction-examples/MRTK_Examples_Package_MRFT.png" width="550" alt="Example Package 1"><br/>

在 Unity 中，使用菜单 **窗口 > Project > 自定义程序包管理器 >** ，并选择 **混合现实 Toolkit 示例**。 

<img src="features/images/hand-interaction-examples/MRTK_Examples_Package_2.png" width="300" alt="Example Package 2"><br/>

在面板右侧的列表中，单击 "导入到" 示例场景名称旁的 "**导入 Project** " 按钮。  例如，可以单击 "HandTracking" 旁边 **的**"**导入到 Project** " 按钮。 

<img src="features/images/hand-interaction-examples/MRTK_Examples_Package_3.png" width="650" alt="Example Package 3"><br/>

导入后，可以在 " **资产 > 示例** " 文件夹下找到它们。
**HandInteractionExamples** 场景是开始体验 MRTK 的空间交互和 UI 构建基块的好地方。

<img src="features/images/hand-interaction-examples/MRTK_Examples_Package_4.png" width="650" alt="Example Package 4"><br/>

### <a name="directly-downloading-and-importing-packages-from-github"></a>直接从 GitHub 下载和导入包

如果不使用混合现实功能工具，则可以直接下载并导入 **Toolkit MixedReality。unitypackage** 从 [MRTK GitHub 的发布页](https://github.com/microsoft/MixedRealityToolkit-Unity/releases)

使用 **资产 > 导入包 > 自定义包** 菜单导入 unitypackage。 导入后，你将能够在 **资产 > MRTK > 示例 > 演示** 中查找示例场景。

<img src="features/images/hand-interaction-examples/MRTK_Examples_Package_Manual1.png" width="650" alt="Example Manual 1"><br/>
