# <a name="keyboard"></a>键盘

![键盘](images/UX_Hero_Keyboard.jpg)

混合现实应用程序可以随时调用系统键盘，这将根据目标平台的功能进行。 例如，HoloLens 2 上的键盘支持直接交互，而 HoloLens (第一代) 键盘支持注视和分流。
