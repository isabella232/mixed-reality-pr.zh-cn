# <a name="keyboard"></a>键盘

![键盘](images/UX_Hero_Keyboard.jpg)

混合现实应用程序可以随时调用系统键盘，这根据目标平台的功能运行。 例如，HoloLens 2键盘支持直接手部交互，而HoloLens (第一代) 键盘支持凝视和敲击。
