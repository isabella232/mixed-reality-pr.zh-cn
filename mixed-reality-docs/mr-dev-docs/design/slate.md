# <a name="slate"></a>平板

![平板](images/UX_Hero_Slate.jpg)

平板电脑是一种精简窗口样式控件，用于显示纯文本、图像或文档等 2D 内容。 平板电脑提供可抓取的标题栏，其按钮用于"关注我 (标记和) "和关闭功能。 平板电脑中的内容可以通过直接操作进行滚动 [，](direct-manipulation.md#2d-slate-interaction) 就像物理触摸屏一样。 也可使用凝视、敲击和拖动手势或运动控制器的指针。
