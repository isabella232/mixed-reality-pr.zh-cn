# <a name="slider"></a>Slider

![Slider](images/UX_Hero_Slider.jpg)

滑块是一个 UI 组件，用于设置值。 它提供可调整内容的可视指示，以及内容总范围中的当前设置。 它显示为一个轨迹，其中任一侧有选项。 将一个手柄或手柄拖动到一端或另一端以做出选择，指示当前值。 可以通过直接或以距离直接抓取滑块来移动收缩滑块。 滑块可处理各种输入类型，例如凝视、敲击和运动控制器。
