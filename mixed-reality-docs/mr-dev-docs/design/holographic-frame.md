### <a name="large-objects"></a><span data-ttu-id="2135a-120">大型对象</span><span class="sxs-lookup"><span data-stu-id="2135a-120">Large objects</span></span>

<span data-ttu-id="2135a-129">有关针对大型对象进行设计的示例，请参阅[为大型对象设计。](holographic-frame.md#volvo-cars)</span><span class="sxs-lookup"><span data-stu-id="2135a-129">For an example of designing for large objects, see [Volvo Cars](holographic-frame.md#volvo-cars).</span></span>

<span data-ttu-id="2135a-131">许多对象或组件的体验应考虑使用用户周围的完整空间，以避免直接在用户前面混乱全息帧。</span><span class="sxs-lookup"><span data-stu-id="2135a-131">Experiences with many objects or components should consider using the full space around the user to avoid cluttering the holographic frame directly in front of the user.</span></span> <span data-ttu-id="2135a-132">建议将内容引入体验的速度变慢，尤其是对于计划向用户提供许多对象的体验。</span><span class="sxs-lookup"><span data-stu-id="2135a-132">We recommend slowing introducing content to an experience, especially with experiences that plan to serve many objects to the user.</span></span> <span data-ttu-id="2135a-133">关键在于让用户 **了解** 体验中的内容布局，这有助于用户在空间上了解内容更新时周围的内容。</span><span class="sxs-lookup"><span data-stu-id="2135a-133">The key is to **let users understand the content layout** in the experience, which helps them gain a spatial understanding of what’s around them as content updates.</span></span>
