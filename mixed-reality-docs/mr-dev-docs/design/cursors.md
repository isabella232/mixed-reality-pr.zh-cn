<span data-ttu-id="c60ca-106">光标提供持续的反馈，其中基于耳机认为用户当前焦点在给定时间的位置。</span><span class="sxs-lookup"><span data-stu-id="c60ca-106">A cursor provides continuous feedback based on where the headset believes a users current focus is at a given time.</span></span> <span data-ttu-id="c60ca-107">光标反馈包括虚拟环境中的哪些区域、全息图或点对输入的响应。</span><span class="sxs-lookup"><span data-stu-id="c60ca-107">Cursor feedback includes what area, hologram, or point in the virtual environment responds to input.</span></span> <span data-ttu-id="c60ca-108">尽管光标是设备理解用户注意的数字表示形式，但这与确定用户的意图并不相同。</span><span class="sxs-lookup"><span data-stu-id="c60ca-108">Even though the cursor is a digital representation of where the device understands the user's attention to be, that's not the same as determining the user's intentions.</span></span> <span data-ttu-id="c60ca-109">光标反馈还允许用户了解系统对预期的响应。</span><span class="sxs-lookup"><span data-stu-id="c60ca-109">The cursor feedback also lets users know what system responses to expect.</span></span> <span data-ttu-id="c60ca-110">你可以使用反馈将其意图传达给设备，从而提高用户信心。</span><span class="sxs-lookup"><span data-stu-id="c60ca-110">You can use the feedback to communicate their intention to the device, which increases user confidence.</span></span>

## <a name="cursor-customization-recommendations"></a><span data-ttu-id="c60ca-155">游标自定义建议</span><span class="sxs-lookup"><span data-stu-id="c60ca-155">Cursor customization recommendations</span></span>

### <a name="cursor-scale"></a><span data-ttu-id="c60ca-157">游标刻度</span><span class="sxs-lookup"><span data-stu-id="c60ca-157">Cursor scale</span></span>

* <span data-ttu-id="c60ca-178">将光标移到靠近用户的位置时，会将光标锁定到交互元素，从而有助于提高用户在使用选择操作时与该元素交互的置信度。</span><span class="sxs-lookup"><span data-stu-id="c60ca-178">Magnetically locking the cursor to an interactive element when it's close to the user can help improve confidence that user will interact with that element when they use a selection action.</span></span>
